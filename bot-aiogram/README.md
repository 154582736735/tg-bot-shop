# Telegram бот с регистрацией (FSM)

Telegram бот с формой регистрации. Пользователь вводит имя и номер телефона через кнопку.

## Технологии

- Python 3.10+
- aiogram 3.x
- FSM (машина состояний)
- ReplyKeyboardMarkup с запросом контакта

## Команды

/start — приветствие и инлайн-меню
/reg — начать регистрацию
/help — помощь

## Регистрация

1. Пользователь вводит /reg
2. Бот запрашивает имя
3. Бот запрашивает номер телефона через кнопку
4. Бот выводит введённые данные

## Установка и запуск

1. Установите зависимости:
   pip install -r requirements.txt

2. Создайте config.py:
   TOKEN = "токен_бота_от_BotFather"

3. Запустите бота:
   python run.py
