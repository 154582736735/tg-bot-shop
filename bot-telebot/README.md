# Telegram бот для заказа еды

Telegram бот с корзиной, кулдауном и автоподтверждением заказа.

## Технологии

- Python 3.10+
- pyTelegramBotAPI (telebot)
- SQLite3 (кулдаун между заказами)

## Функционал

- Выбор категорий и товаров через инлайн-кнопки
- Добавление в корзину
- Расчёт суммы заказа
- Кулдаун 10 минут между заказами
- Автоматическое удаление сообщений бота
- Отправка заказа администратору
- Чёрный список пользователей (кулдаун)

## Команды

/start — начать работу

## Установка и запуск

1. Установите зависимости:
   pip install -r requirements.txt

2. Вставьте токены ботов в переменные bot и bot1

3. Запустите скрипт:
   python bot.py

## Структура

- bot.py — основной файл с логикой
- users.db — база данных с кулдаунами (создаётся автоматически)
